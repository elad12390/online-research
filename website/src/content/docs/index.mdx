## Quick Start

```bash
# Clone and start with Docker
git clone https://github.com/elad12390/online-research.git
cd online-research
./setup.sh
```

Open [http://localhost:3000](http://localhost:3000) and start researching!

## Multi-Provider Support

Research Portal supports multiple AI providers:

| Provider | Models | Best For |
|----------|--------|----------|
| **Anthropic** | Claude Sonnet 4.5, Opus 4.1 | Complex research, nuanced analysis |
| **OpenAI** | GPT-4o, GPT-4 Turbo | General research, coding tasks |
| **Google** | Gemini Pro | Fast iterations, multimodal research |

## Architecture

The portal uses a **two-system architecture**:

1. **Research Portal** - File-based project browser with live updates
2. **Research Wizard** - Database-driven AI agent orchestration

Both systems work together to provide a seamless research experience from creation to consumption.
