CRAAP FRAMEWORK (For EVERY source):

âœ“ Currency: Is it current enough for this topic?
âœ“ Relevance: Does it directly answer your question? Intended audience?
âœ“ Authority: Who wrote it? Credentials? Affiliation? Peer reviewed?

SIFT METHOD (Quick triage):

1. STOP - Pause before sharing or trusting. Take a breath.
2. INVESTIGATE the source - Who publishes? Check author, organization, past work.
3. FIND better coverage - Search claim elsewhere; look for reputable outlets or primary sources.
4. TRACE - Follow claims to original context: studies, reports, datasets.

LATERAL READING (Expert technique):

- Leave the page to check what others say about the site/author
- Open new tabs: search "site:domain.com" + about, or author name + affiliation
- Look for corroborating or contradicting evidence
- This is MORE effective than scrutinizing the page itself

For EVERY major claim:

1. TRACE to primary source:
   - Collect citation details (author, year, title, DOI)
   - Search Google Scholar, CrossRef, or PubMed using title or DOI
   - If only paraphrase exists, search distinctive phrases in quotes

2. When you find the paper:
   - Read abstract and methods section
   - Check sample size, study design, outcome measures, limitations
   - Verify headline/summary accurately reflects study conclusions
   - Check if preprint vs peer-reviewed, look for retractions/corrections

3. Cross-check with 2+ independent sources
   - Find at least 2 other sources that support or refute the claim
   - Note when information comes from single source only

4. Check methodology:
   - Sample size (is it adequate?)
   - Study design (RCT > observational > case study)
   - Conflicts of interest or funding bias

5. Verify statistics and charts:
   - Find underlying dataset if possible (supplementary files, data repositories, gov data)
   - Check axis labels, scales, units in charts
   - Watch for truncated axes and selective time windows
   - Recalculate simple statistics when feasible

6. Use fact-checkers:
   - PolitiFact, Snopes, AFP Fact Check for public claims
   - Reverse image search (Google Images, TinEye) for image origin
